# 4. Password Generator

### Overview
This Python project generates secure random passwords based on configurable criteria.

The program ensures that the password contains required character types such as numbers and special characters.

### Features
Customizable password length  
Optional numbers  
Optional special characters  
Secure random generation  
Password requirement validation  

### Technologies
Python  
Random module  
String utilities  

### How It Works
The script randomly selects characters from allowed character sets until the password satisfies all required conditions.

### Example Output

```
Your password is: A7!k9@LpQ2
```

### Learning Goals
Secure password creation  
Randomization techniques  
Input validation logic
