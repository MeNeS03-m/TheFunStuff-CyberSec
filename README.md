# Cybersecurity and Networking Mini Projects

This repository contains several small projects focused on cybersecurity, networking, and Python automation.  
The goal of these projects is to practice security concepts, scripting, and basic networking techniques while building a practical portfolio.

These tools demonstrate topics such as password security, network discovery, IP intelligence gathering, automation, and authentication testing.
